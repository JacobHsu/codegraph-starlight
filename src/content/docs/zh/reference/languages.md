---
title: 語言支援
description: CodeGraph 解析的所有語言及其識別的副檔名。
---

語言支援依副檔名自動識別，無需任何設定。

| 語言 | 副檔名 | 狀態 |
|---|---|---|
| TypeScript | `.ts`、`.tsx` | 完整支援 |
| JavaScript | `.js`、`.jsx`、`.mjs` | 完整支援 |
| Python | `.py` | 完整支援 |
| Go | `.go` | 完整支援 |
| Rust | `.rs` | 完整支援 |
| Java | `.java` | 完整支援 |
| C# | `.cs` | 完整支援 |
| PHP | `.php` | 完整支援 |
| Ruby | `.rb` | 完整支援 |
| C | `.c`、`.h` | 完整支援 |
| C++ | `.cpp`、`.hpp`、`.cc` | 完整支援 |
| Swift | `.swift` | 完整支援 |
| Kotlin | `.kt`、`.kts` | 完整支援 |
| Scala | `.scala`、`.sc` | 完整支援（類別、trait、方法、型別別名、Scala 3 enum） |
| Dart | `.dart` | 完整支援 |
| Svelte | `.svelte` | 完整支援（script 提取、Svelte 5 rune、SvelteKit 路由） |
| Vue | `.vue` | 完整支援（script + script-setup、Nuxt 頁面/API/中介軟體路由） |
| Liquid | `.liquid` | 完整支援 |
| Pascal / Delphi | `.pas`、`.dpr`、`.dpk`、`.lpr` | 完整支援（類別、record、介面、enum、DFM/FMX 表單） |
| Lua | `.lua` | 完整支援（函式、方法、局部變數、`require` 匯入、呼叫邊） |
| Luau | `.luau` | 完整支援（Lua 全部功能，加上型別簽章、`type` 別名、Roblox `require`） |
